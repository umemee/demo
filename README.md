📢 [AI 지원사업 매칭 시스템] 데모 버전 최초 실행 가이드 (필독)
======================================================================

본 프로그램은 인터넷 클라우드 서버로 데이터를 전송하지 않고, 
사용자분의 컴퓨터 내부에서 "고성능 로컬 AI"를 직접 구동하는 보안 SaaS 프로그램입니다.
정상적인 구동을 위해 아래의 [준비 단계]부터 순서대로 진행해 주세요.

----------------------------------------------------------------------
📁 [준비 단계] 전용 작업 폴더 만들기 및 프로그램 다운로드
----------------------------------------------------------------------

1. 바탕화면에 [새 폴더]를 하나 만들고, 폴더 이름을 'AI_매칭시스템'으로 지정합니다.
2. 배포 링크(GitHub Releases)에서 [app.exe] 파일을 다운로드합니다.
3. 다운로드된 [app.exe] 파일을 방금 바탕화면에 만든 'AI_매칭시스템' 폴더 안으로 이동시켜 줍니다.

----------------------------------------------------------------------
🏃‍♂️ [최초 1회] 딱 3분 만에 AI 두뇌 환경 세팅하기
----------------------------------------------------------------------

1. AI 엔진(Ollama) 설치하기
   - 공식 홈페이지(https://ollama.com)에 접속합니다.
   - 메인 화면의 [Download for Windows] 버튼을 눌러 설치 파일을 다운로드합니다.
   - 다운로드된 'OllamaSetup.exe' 파일을 실행하여 설치를 완료해 주세요.

2. 9GB 고성능 AI 모델 파일 다운로드 및 이름 변경
   - 아래의 허깅페이스 공식 저장소 주소에 접속합니다:
     https://huggingface.co/unsloth/gemma-4-E2B-it-GGUF/blob/main/gemma-4-E2B-it-Q4_K_M.gguf
   - 화면 중앙의 [download] 버튼을 눌러 9GB 용량의 모델 파일을 다운로드합니다.
   - ⚠️ 중요: 다운로드가 완료되면 파일 이름을 반드시 아래와 같이 똑같이 변경해 주세요.
     - 바꿀 파일 이름: gemma4.gguf
   - 이름이 바뀐 [gemma4.gguf] 파일을 바탕화면의 'AI_매칭시스템' 폴더(app.exe가 있는 곳)로 이동시켜 줍니다.

3. Ollama 엔진에 'gemma4' 모델 수동 등록하기
   - 바탕화면의 'AI_매칭시스템' 폴더를 엽니다.
   - 폴더 안의 빈 공간(아무 파일도 없는 곳)에서 키보드의 Shift를 누른 채 마우스 우클릭을 합니다.
   - 메뉴 나오는 것 중 [여기에 PowerShell 창 열기] 또는 [터미널에서 열기]를 선택합니다.
   - 파란색(또는 검은색) 창이 뜨면 아래의 두 명령어를 한 줄씩 복사해서 입력하고 각각 Enter를 누릅니다:

     (1) echo "FROM ./gemma4.gguf" > Modelfile
     (2) ollama create gemma4 -f ./Modelfile

   - 문장 처리가 완료되면서 'success'라는 문구가 뜨면 등록이 성공한 것입니다. 
     이제 열려 있는 터미널 창을 닫으셔도 좋습니다.

----------------------------------------------------------------------
🚀 [상시 실행] 프로그램 기동 및 사용 방법
----------------------------------------------------------------------

- 세팅이 끝났다면, 'AI_매칭시스템' 폴더 안에 있는 [app.exe] 파일을 더블 클릭하여 실행합니다.
- 자동으로 검은색 백엔드 서버 창이 켜지며 시스템이 시동됩니다. (창을 절대 닫지 마세요)
- 웹 브라우저(Chrome/Edge 등)를 열고 주소창에 아래 주소를 입력하여 접속합니다:

  http://localhost:4173

- [화면 기능 안내]
  1. PDF 업로드: 회사의 사업계획서나 소개서 파일을 드래그하면 AI가 정밀 분석을 시작합니다.
  2. 과거 기록 로드: 메인 화면 우측 상단의 [과거 Case ID 입력] 칸에 
     'upload_20260527_083226_654'와 같은 기존 ID를 넣고 [기록 로드]를 누르면, 
     서버 장부에서 이전 분석 리포트 카드가 1초 만에 그대로 복원됩니다.

----------------------------------------------------------------------
🔄 [비하인드] 원격 자동 업데이트 시스템 안내
----------------------------------------------------------------------

- 코드가 개선되어 최신 버전이 나오면, 프로그램이 스스로 'AI_매칭시스템' 폴더 안에
